노트 생성기

이 작업은 자바스크립트의 p5 리듬게임과 연관있는 작업물입니다.
노트 생성기에서 곡을 선택한 뒤, 노트 재생을 누르면 음악이 재생됩니다.
노트 생성을 누르면 텍스트 입력창이 생성되며, 그곳에 키보드의 "s,"d","f"," ","j", "k", "l"을 타이밍에 맞게 입력하면 그 내용이 저장됩니다.
음악 재생이 끝나면 노트 생성 버튼은 노트 반영 버튼으로 바뀌게 되고, 이를 누르면 왼쪽 창에 생성한 노트가 표시됩니다.
이 상태에서 노트 재생을 누르면 음악이 다시 재생되면서 스크롤 바가 자동으로 이동하게 되어야 합니다만,
역량 부족의 문제로 오작동이 심해, 음악만 재생될 확률이 높습니다.자세한 내용은 동영상에서 자세히 나와있으며, 성공하는 모습은 동영상의 후반부에 나옵니다.
마지막으로 생성한 노트를 내보내기 버튼을 클릭하면, 파일을 저장할 수 있습니다. 가급적이면 .파일명 끝에 txt를 붙이는 것을 추천합니다.
이렇게 생성된 노트 파일은 자바 스크립트 탭의 p5 리듬게임에서 사용할 수 있습니다.
